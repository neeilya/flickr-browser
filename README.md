# Flickr-browser

> Simple flickr web browser built with Vue  
> Live demo here: https://smartdevs.io/flickr

## Technologies used
- Vue (2.2)
- Webpack (2) + vue-loader
- Eslint Airbnb
- Axios
- lodash/debounce
- Bulma css framework
- Sass preprocessor

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
